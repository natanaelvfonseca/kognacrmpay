# Kogna CRM & Pay

Ambiente demonstrativo com CRM comercial e gestão financeira integrados. Todos os dados são fictícios e o projeto não utiliza banco de dados, APIs privadas ou credenciais dos sistemas originais.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build para Vercel

```bash
npm run build
```

## Módulos

- `/crm`: pipeline comercial, indicadores e leads fictícios.
- `/pay`: turmas, recebíveis, cobrança e risco financeiro fictícios.
